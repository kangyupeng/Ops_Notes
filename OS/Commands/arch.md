# arch 



## 作用：

用来查看系统硬件架构（CPU指令集），就是常说的 i386/i486/i586/i686...



## 语法

```txt
SYNOPSIS
       arch [OPTION]...

DESCRIPTION
       Print machine architecture.

       --help display this help and exit

       --version
              output version information and exit
```



## 应用

查看系统硬件架构：

```bash
[root@apt-1 ~]# arch 
x86_64
```
